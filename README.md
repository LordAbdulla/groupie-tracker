# Groupie Tracker

Groupie Tracker is a Go web application that interacts with the Groupie API to display detailed information about artists, bands, their members, creation dates, albums, concert locations, and relations between all data points.  
The project focuses on API handling, JSON parsing, data structuring, error management, and dynamic rendering using HTML templates.

---

##  Features

- Fetches and parses data from the official Groupie Tracker API:
  - Artists
  - Locations
  - Dates
  - Relations
- Displays artist details in a clean web interface
- Shows members, first album date, creation date, and more
- Visualizes upcoming and past concerts
- Handles all errors (404, 400, 500, wrong endpoints)
- Fast and minimal backend using Goâ€™s standard library
- Clean HTML/CSS frontend
- Proper routing and template rendering
